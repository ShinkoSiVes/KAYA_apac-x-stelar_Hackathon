# KAYA App

Application code for the KAYA mobile budgeting copilot.

## Folder structure
- `assets/` — static assets
- `bl/` — business logic
- `config/` — configuration
- `controllers/` — request / flow handlers
- `helper/` — utilities
- `model/` — data models
- `scripts/` — build / tooling scripts
- `views/` — UI screens
