# KAYA — "Kaya mo 'to."

AI budgeting copilot for Filipino students and young professionals, built on Stellar.

**Track:** Payment & Consumer Applications · APEC x Stellar Hackathon 2026

## What it does
Fund in (GCash/Maya) → AI coaches every purchase → Save it (Soroban vault) or Pay it (QRPh, gasless) → build on-chain reputation.
